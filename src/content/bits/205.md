设置EDL：

Output Format 用File_32即可。 Tracks记得把不用选的轨道都拿掉，List Options里面 Include in List 选项只留Clip Names，必要情况下可以留一下Source File Name（如果剪辑爱改名），还可以留一个Frame Count，其他的全不选。

Preview之后把所有文字部分复制粘贴到Excel第二列，第一列做一个数值行出来用来之后排序。

现在的EDL里“From Clip”和其他内容是夹杂在一起的，先用Sort把这两部分分开。Sort第二列，把Clip记数开头的部分全都挪到第三列去，然后重新用第一列排回正序。

之后可以做三件事：1. 把B列的From Clip Name用冒号分隔开（Data - Text to Column），方便查询Clip Name；2. 同理把C列的信息用空格分隔开；3. 把B列整体向上拉一或两行，直到Clip Name和时间码对准。

用H列（或G列，总之就是时间码的第一列）做一个Sort，一方面把那些错出去一列的时间码都拉回来，另一方面把0000开头的渐变啥的都拿掉。借此也可以把空行全都拿掉。

用Clip Name再Sort一下，集中拿掉所有不是Archival的clip，包括访谈和Temp Music。

检查一下，剩下的就是一些手调的工作了，就是一些细部错行，耐心即可。
