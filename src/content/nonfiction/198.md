2020年6月18日，洛杉矶没有新闻
（本文不是标准意义上的非虚构作品，有大量未查证/无法确认/推测的内容。请不要当成新闻报道阅读。）

<!--more-->

2020年6月18日，美国洛杉矶市，天气阴。漫天都是云朵，像是巨大的灰色铁幕，罩在整个洛杉矶的上空。
艾琳娜戴好口罩、橡胶手套，又把消毒喷雾和桶装消毒湿巾放进车里。车里还有一把折叠沙滩椅，后窗下面放着一件开身牛仔上衣，领口处放着一张照片。
她摸了摸那件牛仔上衣，说，“宝贝。”
她回屋子去给狗牵狗绳，三只布鲁塞尔格里芬犬团团绕在她的脚边疯狂地晃着尾巴。她抱起那只看上去最有主意的Elvis的前腿，把狗绳拴好之后另一手拿上了它的小床。
她的身形被灰色的狗床衬得格外瘦小，步子却格外笃定。
一切准备停当后，她发动汽车。艾琳娜已经有好几年不听音乐了，她打开车上的新闻广播，听了没两句又关掉。她也不需要交通导航，因为这条路她几乎每天都要走一遍——从她位于影视城的家前往好莱坞永恒墓地。
从曾经是她的独生子安东·尤金的家，前往如今他睡着的地方。

<br/>

2016年6月18日晚，是安东一位多年好友的生日聚会。
他们的乐队本打算在当晚的晚些时候聚在一起排练——或者说，玩些曲子。安东蓬勃无羁的创造力在音乐上同样有着体现，特别是当他们的乐队随着年龄增长渐渐从摇滚转向了车库朋克风格之后，曲风的束缚仿佛一层层被剥离掉，而更加直接地传达出他们的想法。
但是安东没有来。在18日夜到19日凌晨的某个时间，他们开车去了他的家，山路上没有一盏灯，黑得伸手不见五指，静得令人发慌。
后来，他们打电话叫来了安东的父母。

<br/>

那一晚后数月，纪念安东的安东·尤金基金会宣告成立，进行日常执行维护的是那夜过生日的戴夫。截至今日，基金会已经帮助了近十位身患疾病或残疾的艺术家，更向南加州大学下属的囊肿性纤维化诊所捐款一百万美元，大幅度增加了诊所的每周接诊容量，为更多患者增加了生的希望。
同样是在那件事发生之后三四个月，一同去他家找他的安东好友瑞恩和他的女朋友确定了关系。他曾经一度因为这起事故而饱受心理折磨——对于少年丧父的他来说，安东一直以来的陪伴和扶持非同寻常。甚至有很长一段时间，他都没办法再度打开那些和安东在一起的合照。
时光荏苒，他和他的女友至今感情稳定，经常在社交网络上发布甜蜜的动态。瑞恩在他们确定关系三周年时对她告白说，“你是我的珍宝”。

<br/>

他们都参与了安东纪录片的摄制。
纪录片《爱你，安东》在2019年中登陆欧美多地的独立院线，影片聚集了六十多位受访人，其中不乏一线明星、导演、行业内知名的工作人员，票房可称惨淡，却并不出人意料。
独立纪录片向来不以盈利为目标，对于更多人来说，“每看一次，我就觉得我和我的儿子多待了九十分钟。”安东的父亲维克多说。
但在颁奖季密集的“供您参考”海报争夺战和密集的免费观片会中，《爱你，安东》还是义无反顾地跳了进去。
剧组里有人这样说：“对于拿奖，我们其实并不抱太大期望。但是能够多让一些人看到他的故事，也是好的。”
这部电影同样没有入围奥斯卡最佳纪录长片的角逐，在2020年颁奖前一个月官方公布的十部纪录长片的“短名单”中名落孙山。
一年又一年，奥斯卡金像奖的选择标准越来越远离个人化的人生体验，转而拥抱对社会的思考、对世界的疑问，反映真实世界的纪录片奖项更加如此。纵然奥斯卡九十周年的纪念混剪中安东的镜头紧紧接在同样英年早逝的美国电影传奇詹姆斯·迪恩之后，这样的青睐也没办法给得更多。

<br/>

安东的母亲停好了车，牵着他的宠物狗Elvis走向他的雕像。艾琳娜总是会握一握被加州的阳光烤热的雕像的手，但是疫情期间，她先掏出了酒精喷雾开始消毒。
新冠疫情开始在美国蔓延的时候，她是除了华人之外最先戴上口罩的那群人。在她经常去的俄罗斯食品商店，一度有很多人拿看怪物的眼神瞟着她，但她毫不在意。
“那些人都是傻子，我得关心我自己。我才不在乎他们怎么看我！”她说，带着俄罗斯口音的英语里是她这样的女人会带着的那种坚定和梗倔。亚马逊上一度买不到口罩，她得学着一个N95用几次，以及询问他人用酒精擦口罩是否有效。她同样像很多中国老人一样笃信偏方，每天喝大量的柠檬水。
也是在那段时间，老两口学会了使用在线买菜送货上门业务，戴夫有时也会买一些菜，放在安东家门口。
然而即使如此，她和维克多仍然每天轮流前往好莱坞永恒墓地，陪他们的儿子待上一会儿。在洛杉矶封城最严格的那段时间，墓地管理彻底禁止了游客来访，只允许逝者亲属进入，于是那里几乎成了全洛杉矶最安全的地方。
对他们来说，每天一趟的墓地之行仿佛是世界的巨大动荡面前唯一的不变。

<br/>

随着洛杉矶防疫政策的放松，偶尔他们也会遇到一些访客。
有一位看上去大略四十岁上下的女士注意到了艾琳娜，惊讶地站在离她两米左右的“社交距离”。
“您是安东的母亲吗？我真是太荣幸能遇见您了！我昨天刚刚看完《爱你，安东》……优秀的电影，可爱的人……谢谢您，真的谢谢您，抱歉因为疫情我没办法拥抱您……”
艾琳娜对她说了谢谢。那女士带来了一捧很漂亮的花，放在雕像前面。

<br/>

2020年6月18日的洛杉矶没有新闻，因为2020年的新闻已经太多。新冠疫情在美国尚未得到缓解，非裔美国人乔治·弗洛伊德的死就把在人群中挤压已久的愤怒、不安、恐惧和压抑一齐引爆。新冠已经被遗忘，越来越多的人走上街头，有的人互相拥抱着发出自己的声音，有的人趁火打劫，点燃一个又一个商铺，每个人都愤怒，但没有人知道这一次愤怒是否能够真的带来改变。几种声音在舆论场中交缠厮杀各不相让，外人看到的只有最丑陋的那一个小小侧面。
在这种事情面前，安东曾经最爱去逛的Amoeba搬迁似乎已经无足轻重。深居好莱坞腹地、紧邻着在《好莱坞往事》中出现的Arclight影院，占地两层的历史性碟店Amoeba由于地段开发，在2019年底便放出风声要进行举家搬迁；在新冠疫情和各地游行的大背景下，盛极一时如Amoeba也不得不低调搬迁，前往著名的日落大道。
Amoeba在社交网络上发布了数日的旧日回忆照片，无数音乐界的大佬、流行文化的符号们在凌乱的Amoeba一楼那小小的舞台上点燃了整个场子，而这些都将默默地成为过去。官方账号不无遗憾地说，“它值得一个比现在更好的告别。”
毕竟，从Amoeba下楼去地下停车场的楼道里贴满了的、不知积攒了多少年也不知道有没有安东一份的密密麻麻的贴纸，是搬不走的。
Amoeba已经算得上幸运。Santa Monica几家安东爱去的饭馆、纽约SoHo区他曾热爱欣赏过的艺术品商店都未能逃过这次的劫难，被砸毁或洗劫。艾琳娜已经不再听每天越发雪上加霜的新闻，每一个熟悉的名字里都是他过去的影子。
她收拾好一切，给Elvis擦干净了四爪，给狗床的底部消了毒，临走时又握了握雕像的手。她踏上石阶，从背后给了她的儿子一个拥抱，启动了车子。

<br/>

安东离开后的第四年，时光的洪流越来越无可阻挡地改变着一切、冲刷着一切。
美国总统由当时的巴拉克·奥巴马换成了唐纳德·特朗普，并且即将完成他的第一个任期。伴随着美国总统沧桑巨变的还有更多更多的东西：漫威宇宙的一个大阶段随着钢铁侠的牺牲而落幕，韩国人第一次拿到了奥斯卡的最高荣耀，女性群体、性少数群体、有色人种群体的故事越来越多地被听见。一场突如其来的疫情把盛夏的人类文明猛地推进秋天，萧杀的西风和经济下行、孤立主义、种族主义一起猛烈地吹刮着，似乎要把社会彻底撕裂。
在那之后，安东·尤金的名字和面孔仍然不断地被提起、被记忆、被缅怀，但在新冠疫情汹汹而来的2020年，当全人类都在经历着集体的变革，当欧美多地争取黑人权益的运动如雨后春笋一般兴起、又引发着各种各样的争议和讨论时，安东和他那个时期的电影在这样一层层注意力的冲刷里，仿佛一场无忧无虑的少年时代，盛夏假期的旧梦。

<br/>

艾琳娜回家，艾琳娜洗过了手，把出门的衣服换下来，打开冰箱。
冰箱里曾经堆满了安东爱吃的火鸡胸肉和吐司，以及寿司外卖里送的酱油，但现在冰箱里的内容显然要健康得多。
安东极其爱吃她做的鸡汤面，但她已经三年多不再下厨，直至最近。她说，“维克多就只会做那几样菜，还是我来吧。”
于是她拧开灶台上的火，在打火石的哒哒响声之后，蓝色的火苗跳跃出来，装满水的锅子放了上去。抽油烟机的声音、水沸的声音、三条狗在她脚边吠叫的声音响成一片，洗碗池上方的小窗台上安东的照片正在朝她微笑。
2020年6月18日，洛杉矶唯一的新闻：艾琳娜·尤金开始做饭。
